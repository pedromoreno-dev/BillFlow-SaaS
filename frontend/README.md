# 💻 BillFlow Frontend | Vue 3 + TypeScript

Este es el cliente web de la plataforma BillFlow. Está diseñado para ser ligero, modular y altamente reactivo.

## 🛠️ Stack Técnico
- **Framework:** Vue 3 (Composition API)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **State Management:** Pinia
- **Internationalization:** vue-i18n
- **Icons:** Lucide Vue

## 📁 Estructura de Carpetas
- `/src/components`: Componentes UI reutilizables (Bento cards, inputs, modals).
- `/src/views`: Páginas principales (Dashboard, Invoices, Clients).
- `/src/composables`: Lógica de negocio reutilizable (Hooks).
- `/src/locales`: Archivos de traducción (ES/EN).
- `/src/services`: Clientes API para comunicación con el Backend.
- `/src/lib`: Utilidades y configuraciones (p. ej. utils.ts).
